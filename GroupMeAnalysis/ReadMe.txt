This console app analyzes data provided by GroupMe to figure out stats about users and top messages.
Download data at: https://web.groupme.com/profile/export
 And check "Message Data"
 Groupme allows downloading a ZIP of conversation data which needs to be unzipped to a folder
The console will prompt for the complete path to the unzipped folder

Metrics provided are Messages per user
Likes per users
Top message recaps (Lists all messages that are tied for most likes or 1 like lower)
Total messages
First message received
